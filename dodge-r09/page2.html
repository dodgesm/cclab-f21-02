Bluebird


Description

Bukowski uses "Bluebird" to describe the speaker's relationships with thier emotions. The bluebird in the poem represents the speaker's kinder and more gentle emotions that they hide behind a hard exterior. According to Bukowski, society is structured to see these softer emotions as vulnerability that should be oppressed.
